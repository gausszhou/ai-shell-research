# AI SHELL GOLANG

```markdown
使用 golang 构建一个 cli 程序，作用是在用户和命令解释器之间建立一座桥梁，要求如下
1. 使用和当前相同的 shell 执行用户的命令 
2. 需要能持续性的交互
3. 使用我们自己的提示符
4. 在我们自己的提示符之前加上 [AI SHELL] 标签 
5. 工作目录要显示完整路径
6. 工作目录要能同步更新展示
7. 不要做任何多余的功能拓展
8. 支持 cmd powershell bash 等命令解释器
9. 支持 conhost openconsole(windows terminal) gnome terminal 等终端  
```
